Wiggly Border Generator

This generator creates organic, hand-drawn style borders using a single continuous SVG path. The path is built by generating points along each edge with varying wave offsets, then connecting them with smooth Catmull-Rom curves converted to Bezier curves. The result scales naturally with any container size while keeping the border stroke consistent using vector-effect="non-scaling-stroke".
